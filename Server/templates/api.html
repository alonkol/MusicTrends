MusicTrends API

  • /api/categories - Get all music categories
  • /api/songs/likes/top/[amount]
  • /api/songs/dislikes/top/[amount]
  • /api/songs/views/top/[amount]
  • /api/songs/views/bottom/[amount]
  • /api/words/top/[amount]
  • /api/words/bottom/[amount]
  • /api/songs/wordscore/top/[amount]
  • /api/artists - Get all artists
  • /api/songs_for_artist/[artist] - Get all songs for artists
  • /api/lyrics/get?song=[song]&artist=[artist] - Show lyrics for song
  • Only For Authorised Personal
  • /api/youtube/update?key=[key]&song=[song]&artist=[artist] -Update Youtube Data for song
  • /api/songs/add?key=[key]&song=[song]$artist=[artist]&category=[category] - Add song to DB
  • /api/blacklist_artist?key=[key]&artist=[artist] - Change artist status to inactive
  • /api/lyrics/update?key=[key]&song=[song]$artist=[artist]&lyrics=[lyrics] - Update lyrics
